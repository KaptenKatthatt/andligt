# Publish Room Checklist

## Purpose

This checklist must be completed before a new room is published.

The goal is not to publish quickly.

The goal is to publish something worth returning to.

---

# 1. The Question

- [ ] The room is built around one clear human question.
- [ ] The question is specific and well-defined.
- [ ] The room does not attempt to answer multiple unrelated questions.
- [ ] The question feels timeless rather than tied to current events.

---

# 2. The Core

- [ ] There is one clear central idea.
- [ ] The opening creates curiosity without explaining everything.
- [ ] The core text can be read comfortably in about five minutes.
- [ ] The room still feels complete even if the user skips the deeper sections.

---

# 3. Structure

- [ ] Short opening
- [ ] Core reflection
- [ ] Reflection questions
- [ ] Historical context (when relevant)
- [ ] Original source references
- [ ] Optional deeper reading

---

# 4. Reading Experience

- [ ] The room feels calm.
- [ ] The text is not overloaded.
- [ ] There is enough whitespace between sections.
- [ ] The user has time to pause and think.
- [ ] Nothing encourages rushing.

---

# 5. Editorial Review

- [ ] The room follows the Manifest.
- [ ] The room follows the Content Philosophy.
- [ ] The room follows the Editorial Guidelines.
- [ ] The room respects the source material.
- [ ] The room does not feel like an AI-generated article.

---

# 6. Final Reflection

Before publishing, answer these questions honestly.

- [ ] Would I enjoy reading this room again in one year?
- [ ] Does this room leave the reader with one meaningful thought?
- [ ] Does this room make Visdomsatlasen better, not merely larger?
- [ ] If this were the only room someone read today, would it have been time well spent?

If any answer is **No**, continue refining the room before publishing.
