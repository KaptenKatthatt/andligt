# Verify Sources Checklist

## Purpose

Every historical statement, quotation and interpretation published in Visdomsatlasen must be traceable to a reliable source.

The goal is not to collect as many sources as possible.

The goal is to ensure that every published room is trustworthy.

---

# 1. Original Sources

- [ ] Every quotation has been verified against the original source or a trusted edition.
- [ ] Book, chapter, verse or section is recorded.
- [ ] The quotation has not been altered in a way that changes its meaning.
- [ ] Any modernization or paraphrasing is clearly identified.

---

# 2. Historical Facts

- [ ] Historical people are correctly identified.
- [ ] Historical events are correctly described.
- [ ] Dates and time periods have been verified.
- [ ] Geographic information has been verified.
- [ ] Claims are supported by reliable historical sources.

---

# 3. Context

- [ ] Quotations are not taken out of context.
- [ ] The surrounding text has been considered.
- [ ] The author's original intention has been respected where possible.
- [ ] Important historical context has not been omitted.

---

# 4. Interpretations

- [ ] Interpretations are clearly separated from facts.
- [ ] Multiple interpretations are mentioned when they are relevant.
- [ ] Areas of historical uncertainty are openly acknowledged.
- [ ] Speculation is clearly identified as speculation.

---

# 5. AI Verification

- [ ] AI has not invented quotations.
- [ ] AI has not invented historical facts.
- [ ] AI has not merged different quotations into one.
- [ ] AI has not presented uncertain information as fact.

---

# 6. Source Quality

For every major source, ask:

- [ ] Is this the closest available source to the original text?
- [ ] Is this source considered trustworthy?
- [ ] Can another reader independently verify this information?
- [ ] Would I feel comfortable defending this source publicly?

---

# 7. Transparency

- [ ] All primary sources are documented.
- [ ] Secondary sources are documented when used.
- [ ] The reader can identify where every important claim comes from.
- [ ] References are complete enough that another person can locate the source.

---

# Final Verification

Before publication, ask yourself:

- [ ] Would I trust this room if I knew nothing about the subject?
- [ ] Could another historian or reader verify every important claim?
- [ ] Have I been honest about what is known, unknown and debated?

If any answer is **No**, continue researching before publishing.
